# The Residency — Application Platform

Full-stack application platform for a startup residency program. Applicants fill out a form, get AI-matched to homes via Claude, answer home-specific questions, and submit.

**[Full documentation →](docs/README.md)**

## Quick Start

```bash
npm install
cp .env.local.example .env.local  # add your keys
npm run dev                        # http://localhost:3000
```

## Stack

Next.js 16 · React 19 · Tailwind v4 · Supabase · Claude Sonnet 4.6 · Playwright
