# Residency Application Platform

A full-stack web application for a startup residency program where applicants fill out a multi-section form, get AI-matched to residency homes via the Anthropic Claude API, respond to personalized home questions, and submit their final application. Includes an admin dashboard for reviewing applications and managing homes.
